# Markov-Chain-Monte-Carlo-in-Air-Traffic-Control-Simulation
An M/M/C queueing-based simulation of airport runway operations was developed to evaluate congestion and efficiency under varying flight arrival rates, service rates, and runway capacities. The model estimates aircraft waiting times and supports infrastructure and operational planning decisions.

## Objective
The main objective is to understand how runway capacity affects aircraft delays, waiting times, and congestion levels under different traffic conditions.

## Methodology
- Modeled the runway system as an **M/M/C queue**.
- Simulated aircraft arrivals and departures using Python.
- Evaluated different scenarios by varying:
  - Aircraft arrival rates
  - Runway service rates
  - Number of available runways

## Key Performance Measures
- Average aircraft waiting time
- Average queue length
- Runway utilization
- System congestion levels
